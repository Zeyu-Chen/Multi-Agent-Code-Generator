# Web Server Template

A flexible Rust web server template that serves as the backend implementation target for AutoGPT. This template provides a foundation for dynamic code generation based on user requirements.

## Quick Start

This template is designed to be used with AutoGPT for code generation. The actual implementation will be generated in `main.rs` based on specific requirements.

Note: Do not modify `main.rs` directly as it is generated code. Any base functionality changes should be made in `code_template.rs`.

### Prerequisites

- Rust (latest stable version)
- Cargo

### Setup & Running

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
cargo build
```

3. Run the server

```bash
cargo run
```

Server will be available at `http://localhost:8080`

## Core Features

### Base Infrastructure (code_template.rs)

- Built with Actix-web framework
- CORS support
- JSON-based persistence
- Thread-safe state management with Mutex
- Basic data structures and traits

### Template Components

- Database trait implementations
- CRUD operation templates
- User management templates
- Request/Response structures
- Common utility functions

## Project Structure

```plaintext
web-server/
├── src/
│ ├── main.rs           // Generated by AutoGPT based on requirements
│ └── code_template.rs  // Base templates and utilities
├── Cargo.toml          // Dependencies and project configuration
└── database.json       // Runtime data persistence
```

## Core Dependencies

- actix-web: Web framework
- actix-cors: CORS support
- serde: Serialization
- tokio: Async runtime
